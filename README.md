#Calculator App

A simple Calculator built with HTML,CSS and JavaScript.

#Features

-Addition (+)
-Subtraction (-)
-Multiplication (*)
-Percentage (%)
-Delete last character (DEL)
-Clear display (C)
-Keyboard support
-Error handling

# Technologies used

-HTML5
-CSS3
-JavaScript

#What I Learned
this project helped me practice:
-DOM selection
-Event Listeners
-Conditional statements (if/else)
-Keyboard events
-String methods (slice)
-JavaScript functions
-Basic error handling
